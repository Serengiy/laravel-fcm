## Laravel fcm

### Installation:

#### 1. Make .env from  .env.example
#### 2. Run
    - composer install
    - npm install
#### 3. Generate laravel key
    - php artisan key:generate
#### 4. Run  
    - ./vendor/bin/sail up -d
#### 5. Run migrations
    - ./vendor/bin/sail artisan migrate
#### 6. Run 
    - vite dev 

After all steps create user account and login. 
App is ready for use
